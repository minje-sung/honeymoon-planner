# プロジェクトルール

## セキュリティ
- .env ファイルは絶対に読み込んだり出力したりしないこと
- APIキーは必ず os.getenv() または python-dotenv 経由で読み込むこと
- APIキーをコードにハードコーディングしないこと
- .env ファイルの内容をチャットに表示しないこと

## コーディングルール
- すべてのコメント・変数名・プロンプトは日本語で記述すること
- パッケージ管理は uv を使用すること (pip は使わない)
- 仮想環境は .venv を使用すること

## Gitルール
- コミットメッセージは必ず**英語または日本語**で記述すること（韓国語禁止）

## プロジェクト概要
新婚旅行プランナー AI エージェント
- フレームワーク: OpenAI Agents SDK
- UI: Streamlit
- 対応言語: 日本語
